Overview

What Morph is

Morph is a custom desktop coaching assistant built to support trading card game decisions through a guided interface instead of a generic chatbot layout.

I designed it as a more product-focused experience with a coaching flow, session structure, and a clean assistant UI that feels purposeful.

This project helped me explore how to combine software functionality, UI design, and product thinking into something that feels like a real tool instead of just a code demo.

Features

What Morph can do

Focused Coaching UI

Built like a desktop tool with a clear coaching layout instead of a generic assistant window.

Guided Chat Flow

Supports turn questions, interaction help, and battle-focused coaching through conversation.

Coach Panel

Includes recommendation space, coaching notes, and active session context.

Session-Based Design

Organized more like a product experience than a simple chat window.
